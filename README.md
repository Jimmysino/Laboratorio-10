Sistema sencillo que simula el catálogo de una biblioteca. Permite guardar libros en un arreglo y buscarlos por su título.
Se implementaron las interfaces Iterador y Agregado  para recorrer el catálogo de libros sin exponer directamente el arreglo interno.
El arreglo se ordena alfabéticamente solo (usando el método de la burbuja) cada vez que se agrega un libro nuevo.
